# dotnet-core-vue-cli

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# build for production with minification
npm run build
```

## Develop

### VSCode/VS2017

Use Debug mode to start up.

### NPM

```
npm run hmr
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
